OQtima Registration Script

Implementation guide for OQtima popup registration script

FOR OQTIMA INTERNAL USE ONLY - This script and implementation guide is restricted to authorized OQtima personnel only

Implementation Guide

Follow these steps to integrate the OQtima registration popup script on your landing page. This implementation provides flexibility by allowing you to use any HTML element as a trigger for the registration popup.

Step 1: Add the Script to Your Page

Add the script tag to the <head> section of your HTML. This allows the script to load early and be available throughout your page:

<script src="https://oqtima.com/scripts/registration-popup-script.min.js" data-api-key="oqtima_prod_reg_iframe_f28e57a3d61c" ></script>
Important: The data-api-key attribute is required for authentication. You must use the API key provided by OQtima. Contact your account manager if you don't have an API key.

Step 2: Add Trigger Elements

Add the data-oqtima-trigger attribute to any HTML element you want to use as a trigger for the registration popup. This can be a button, link, image, or any other clickable element:

Button Trigger Example

Code:
<button data-oqtima-trigger data-lang="en"> Register Now </button>
Result:

Link Trigger Example

Code:
<a href="#" data-oqtima-trigger data-lang="en"> Open Registration Form </a>
Result:
Open Registration Form

Step 3: Understand Available Parameters

The OQtima registration script supports several parameters that can be added as data attributes to your trigger elements:

Parameter Description Required Example
data-oqtima-trigger Marks the element as a trigger for the registration popup Yes data-oqtima-trigger
data-referral-type Referral type for tracking:
  • 12 - IB Referral Link
  • 14 - Campaign Link
 No data-referral-type="12"
data-referral-value Referral value for tracking (IB ID or Campaign ID) No data-referral-value="IB0318050083U"
data-lang Language code for the registration form No data-lang="en"
Supported languages: en (English), fr (French), br (Brazilian Portuguese), vn (Vietnamese), th (Thai), es (Spanish), it (Italian), cn (Chinese Simplified), zh (Chinese Traditional), id (Indonesian), jp (Japanese), my (Malay), ar (Arabic)

IB Referral Link

When using an IB referral link, include data-referral-type="12" and data-referral-value with your unique IB identifier:

Button Example (Japanese)

Code:
<button data-oqtima-trigger data-lang="jp" data-referral-type="12" data-referral-value="IB0318050083U"> IBで登録する </button>
Result:

Link Example (Spanish)

Code:
